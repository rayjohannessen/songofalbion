--------------------------------------------------------------------------------------------------------------
DEBUG------------------------------------------------------------------------------------------------------------
--------------------------------------------------------------------------------------------------------------
1, 2, 3 select city, unit, castle to place on map.
5 - enter select object mode

P - selects the Prydain faction (placement/moving will happen for that faction)
L - selects the Llogress faction

--------------------------------------------------------------------------------------------------------------
GUI--------------------------------------------------------------------------------------------------------------
--------------------------------------------------------------------------------------------------------------
C - Brings up Combat Ability window for currently selected unit
ESC - Closes any open windows, otherwise brings up Menu window

  QUICK BAR (QB)
  ---------
	Left-click - Set unit's attack ability
	Right-click- Set unit's defense ability
	Left-click+drag - move ability (drag off QB to remove from QB), abilities can be swapped as well

Left-click (ability on Combat Ability window) - selects ability, displays ability icon at mouse position
Left-click (on QB with ability previsouly clicked from combat ability window) - adds ability to QB

--------------------------------------------------------------------------------------------------------------
Unit Management/Turn-related-------------------------------------------------------------------------------------
--------------------------------------------------------------------------------------------------------------
Q - End turn
W - goes to next unit if appplicable (unit remains in movement queue)
Space - skips current unit (removing it from movement queue), selects next unit if applicable
Ctrl+D - disbands currently selected unit

--------------------------------------------------------------------------------------------------------------
MOVEMENT/SELECTION-----------------------------------------------------------------------------------------------
--------------------------------------------------------------------------------------------------------------
numpad - all keys except for 5 move units in 8 directions
left-click (on object) - selects it, displays info on GUI, attacks selected enemy on second click
left-click (on empty terrain) - deselects any current enemy first click, your units next click
right-click (on terrain) - moves to selected tile
right-click (on enemy)   - selects, if already selected it will attack if enough stamina remains (includes movement to 				the target's position)

--------------------------------------------------------------------------------------------------------------
MAP VIEW---------------------------------------------------------------------------------------------------------
--------------------------------------------------------------------------------------------------------------
up,down,left,right arrows - scrolls map
move mouse to edge of screen - scrolls map
E - Center on currently selected object
